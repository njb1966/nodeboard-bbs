║ About NodeBoard BBS ║

╚═══════════════════════════════════════════════════════╝

Welcome to NodeBoard BBS, a modern implementation of classic 1990s Bulletin Board Systems. Experience the nostalgia of the pre-internet era with contemporary technology.

Inspired by classic 1990s BBS systems, built with Node.js. NodeBoard brings the community-driven spirit of old-school BBSs to the modern web.

Connect via traditional telnet clients or use our web-based terminal!

║ Method 1: Telnet or SSH Client ║

╚════════════════════════════════════════╝

Connect via telnet or SSH:

$ telnet localhost 2323
$ ssh -p 2222 user@localhost

Recommended Clients:

  • • SyncTERM - Modern BBS terminal (CP437)
  • • PuTTY - Windows telnet/SSH client
  • • NetRunner - Classic BBS client
  • • Terminal - Built-in on Mac/Linux (SSH)

╚════════════════════════════════════════╝

║ Web Terminal - Connect via Browser ║

╚═══════════════════════════════════════════════════════╝

Connect to the BBS directly from your browser using the web terminal. No additional software required! Full ANSI graphics and color support via xterm.js.

» Launch Web Terminal «
Or open in a new tab

Note: The web terminal connects directly to the BBS via WebSocket with full ANSI/VT100 terminal emulation. For an alternative experience, you can also use a dedicated telnet client like SyncTERM or PuTTY.
