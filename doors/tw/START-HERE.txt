                          START HERE!

Got "SHARE wrong version" or SHARE compatibility errors?
Here's the EASIEST solution:

================================================================================
QUICK FIX: Use DOSBox-X
================================================================================

1. Download DOSBox-X from: https://dosbox-x.com/
   (Click "Download" button, get Windows version)

2. Install it
   (Default location is fine: C:\DOSBox-X\)

3. Run from this folder (doors/tw):
   setup-tw-dosbox-x.bat

4. Configure TradeWars in TEDIT:
   - Press C: Set 1 node, LOCAL type
   - Press G: Accept defaults
   - Press B: Select GENERIC
   - Press Q: Save and quit

5. Test it:
   test-tw-dosbox-x.bat

DONE! Game should work perfectly!

================================================================================
WHY DOSBOX-X?
================================================================================

Regular DOSBox:     Good for games, basic DOS
DOSBox-X:          Better DOS compatibility, built-in SHARE, perfect for BBS doors

TradeWars needs SHARE.EXE. Regular DOSBox requires downloading it separately
and dealing with version issues. DOSBox-X has it built-in - just works!
