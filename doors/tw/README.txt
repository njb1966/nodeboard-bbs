TradeWars 2002 - Quick Reference
================================================================================

FIRST TIME SETUP:
  Run: INSTALL-FIRST.bat
  This downloads SHARE.EXE and launches the configuration editor.

COMMON SCRIPTS:
  INSTALL-FIRST.bat  - First-time setup (downloads SHARE.EXE, runs TEDIT)
  setup-tw.bat       - Run configuration editor (TEDIT)
  bigbang-tw.bat     - Generate universe (first time only)
  test-tw.bat        - Test the game
  launch-tw.bat      - Launch game (used by BBS)

CONFIGURATION FILES:
  TEDIT.EXE          - Configuration editor
  dosbox-tw.conf     - DOSBox settings
  TWCFIG.DAT         - Game configuration

DOCUMENTATION:
  QUICKSTART.txt     - Step-by-step setup guide
  README-SETUP.md    - Detailed installation instructions
  SHARE-INFO.txt     - About SHARE.EXE requirement
  TWINSTR.DOC        - Player instructions
  TWSYSOP.DOC        - Sysop guide

TROUBLESHOOTING:
  See QUICKSTART.txt for common issues and solutions

Quick Setup (3 steps):

1. Run: INSTALL-FIRST.bat
   (Downloads SHARE.EXE and opens TEDIT)

2. In TEDIT:
   - Press C: Configure 1 node, LOCAL type
   - Press G: Set game parameters (defaults OK)
   - Press Q: Quit and save

3. Run: bigbang-tw.bat
   (Generate universe - one time only)

4. Run: test-tw.bat
   (Test the game!)

Done! The game is ready.
